# Function Specification: sha256_checksum

Write a Python function `sha256_checksum(filepath: str) -> str` that:

1. Accepting a file path as input.
2. Reads the file in binary mode.
3. Computes the SHA-256 hash.
4. Returns the hexadecimal string representation of the hash.

No external dependencies allowed (use `hashlib`).
Return ONLY the function code.
